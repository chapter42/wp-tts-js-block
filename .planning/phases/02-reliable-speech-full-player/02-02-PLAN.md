**Section 2: splitIntoChunks(text) function (per D-01, D-02, RESEARCH Pattern 1)**

Implement sentence splitting with these specific rules:
- Use regex `/(?<=[.!?])\s+(?=[A-Z\u00C0-\u024F])/` to split at sentence boundaries — requires period/exclamation/question mark followed by whitespace followed by an uppercase letter (handles Dutch abbreviations per RESEARCH Pitfall 2)
- Filter out empty/whitespace-only chunks (per RESEARCH Pitfall 6)
- For sentences longer than MAX_CHUNK_LENGTH (300 chars), sub-split at clause boundaries using `/(?<=[,;:\u2014])\s+/` (per D-02)
- When sub-splitting, accumulate clauses into a buffer until adding the next clause would exceed MAX_CHUNK_LENGTH, then flush the buffer as a chunk
- Return array of non-empty trimmed strings

```javascript
function splitIntoChunks(text) {
  const SENTENCE_END = /(?<=[.!?])\s+(?=[A-Z\u00C0-\u024F])/;
  const CLAUSE_BREAK = /(?<=[,;:\u2014])\s+/;
  const sentences = text.split(SENTENCE_END).filter(s => s.trim());
  const chunks = [];

  for (const sentence of sentences) {
    if (sentence.length <= MAX_CHUNK_LENGTH) {
      chunks.push(sentence.trim());
    } else {
      const clauses = sentence.split(CLAUSE_BREAK);
      let buffer = '';
      for (const clause of clauses) {
        if ((buffer + ' ' + clause).length > MAX_CHUNK_LENGTH && buffer) {
          chunks.push(buffer.trim());
          buffer = clause;
        } else {
          buffer += (buffer ? ' ' : '') + clause;
        }
      }
      if (buffer.trim()) {
        chunks.push(buffer.trim());
      }
    }
  }
  return chunks.filter(c => c.trim().length > 0);
}
```

**Section 3: pickBestVoice(voices, langCode) function (per D-11, RESEARCH Pattern 4)**

Implement voice quality heuristic:
- Filter voices matching langCode exactly, or with underscore-to-hyphen normalization, or matching the language prefix (e.g., 'nl')
- Score each candidate: +10 for exact lang match, +5 per high-quality keyword in name, -5 per low-quality keyword, +2 for non-default voice
- Sort by score descending, return first (highest score)
- Return null if no candidates match

```javascript
function pickBestVoice(voices, langCode) {
  const langPrefix = langCode.split('-')[0];
  const candidates = voices.filter(v =>
    v.lang === langCode ||
    v.lang.replace('_', '-') === langCode ||
    v.lang.startsWith(langPrefix + '-') ||
    v.lang === langPrefix
  );

  if (candidates.length === 0) return null;

  return candidates.sort((a, b) => {
    let scoreA = 0, scoreB = 0;
    if (a.lang === langCode || a.lang.replace('_', '-') === langCode) scoreA += 10;
    if (b.lang === langCode || b.lang.replace('_', '-') === langCode) scoreB += 10;

    const nameA = a.name.toLowerCase();
    const nameB = b.name.toLowerCase();
    for (const kw of QUALITY_KEYWORDS.high) {
      if (nameA.includes(kw)) scoreA += 5;
      if (nameB.includes(kw)) scoreB += 5;
    }
    for (const kw of QUALITY_KEYWORDS.low) {
      if (nameA.includes(kw)) scoreA -= 5;
      if (nameB.includes(kw)) scoreB -= 5;
    }

    if (!a.default) scoreA += 2;
    if (!b.default) scoreB += 2;

    return scoreB - scoreA;
  })[0];
}
```
